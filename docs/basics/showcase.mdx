- 使用 atomFamily 和 localStorage 的待办事项示例 [![在 CodeSandbox 中打开](https://img.shields.io/badge/Open%20in-CodeSandbox-blue?style=flat-square&logo=codesandbox)](https://githubbox.com/pmndrs/jotai/tree/main/examples/todos_with_atomFamily)

  使用 atomFamily 和 localStorage 实现一个待办事项列表。你可以通过点击 `保存到 localStorage` 将你的待办事项列表存储到 localStorage，然后删除你的待办事项列表并尝试通过点击 `从 localStorage 加载` 来恢复它们。
